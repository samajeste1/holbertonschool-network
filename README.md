# Networking Basics

This project covers fundamental networking concepts including:
- OSI Model
- LAN/WAN/Internet
- MAC and IP addresses
- TCP/UDP protocols
- Ports and network tools

## Files

- `basics_0/0-OSI_model` - OSI model questions
- `basics_0/1-types_of_network` - Network types questions
- `basics_0/2-MAC_and_IP_address` - MAC and IP address questions
- `basics_0/3-UDP_and_TCP` - UDP and TCP questions
- `basics_0/4-TCP_and_UDP_ports` - Script to display listening ports
- `basics_0/5-is_the_host_on_the_network` - Script to ping an IP address
